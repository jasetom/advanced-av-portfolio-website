Project 4 - Particle system and 3D object sculpting

This project was inspired by the examples given in class. It demonstrates particle system in 3d space. It also includes a 3D object which can be altered in real-time using controls on the right hand side. The project was created using Three.js library which is an amazing tool that lets your browser to access the GPU of your computer and use OpenGL capabilities to render 3D objects onto the screen.


Controls: drag mouse to navigate around the 3D space, mouse wheel - zoom in/out. Use side bar to control variables in the scene.
