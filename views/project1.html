Project 1 - Audio synthesis

This project demonstrates audio synthesis using Maximillian library. Down below you can find three 30 second audio pieces generated using sine waves, sawn waves, triangle waves, filters and phasors. To generate these sounds methos such as additive synthesis, substraction synthesis and filtering were used.

Example 1.

>

00:00/00:30

Space allien game soundtrack, created using four maxi oscillators. The sound was synthesised using addition and multiplication of sine waves, sawn waves and a triangle wave.
