Project 2 - Interactive audio synthesis

This project demonstrates interactive audio synthesis using FMOsc which is build using Maximillian library. In order to sythesise the sound user is asked to use the settings on the right hand side. When adjusting the settings we will hear different synthesised sounds and it's wave forms shown on the left.
