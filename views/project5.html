Project 4 - Particle system and 3D object sculpting

For my final project, I have built this interactive audiovisual composition using Three.js and Maximillian in JavaScript. It demonstrates sound reactive environment which utilises 3d space to visualise audio. Visuals are built using WebGL implemented via Three.js which is much faster to render in your web browser compared to processing.js or p5.js which I've used in my previous examples. The audio analysis is done using Maximillian sound library for JavaScript. The idea behind this project is to create an audio-visual experience for the selected soundtrack provided by SoulSonic. And so I have achieved it by creating virtual 3d space in which various geometrical shapes, colours, and filters are affected by audio input and merged together to produce visually stimulating experience. Various parameters of graphical elements are manipulated by selected magnitude values of bins returned by the FFT after analysing the soundtrack. I have also used a very basic algorithm for beat detection which is used to control some actions such as applying post-processing shaders in the composition. Another thing that I've incorporated into the project was spectral centroid analysis on the FFT which detects the “brightness” of the sound. The latter is being used to control colour changes in the scene. To sum this all up I would like to say that I have learned a lot about virtual 3D space creation and manipulation using inputs from analysed sound and I can see myself working on similar projects in the future as they can provide a really strong visual experience for the musicians who want to promote their music.
