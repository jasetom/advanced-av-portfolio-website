Project 3 - Audio visualisation using FFT analysis

This project demonstrates audio visualisation possibilities using Fast Fourier Transform (FFT) on an audio file and HTML5 canvas elements with the help of Javascript. All you have to do is chose a song and then browse between 7 varying precoded visualisations of different complexities. Visuals are created using basic geometrical shapes, trigonometry or particle systems. (Works best on chrome!)
